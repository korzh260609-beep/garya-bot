# Project Structure — Архитектурная карта проекта

## 1. Файловая структура (ветка dev)
/index.js        — главный сервер бота, webhook Telegram  
/db.js           — подключение PostgreSQL  
/sources.js      — Sources Layer (скелет)  
/classifier.js   — классификатор задач (скелет)  
/package.json    — зависимости и скрипты  
/WORKFLOW.md     — карта проекта, дорожная карта, прогресс  

---

## 2. Основные слои архитектуры

### 2.1 Transport Layer
- Telegram Webhook  
- Express сервер  
- Приём и маршрутизация входящих апдейтов  

### 2.2 Memory Layer
- Таблица chat_memory  
- Запись сообщений  
- Подгрузка последних N сообщений  
- ❗ Будущее: долговременная память  
- ❗ Будущее: rehydrate контекста  

### 2.3 Sources Layer
- Реестр источников (RSS, web, API)  
- Функции listActiveSources, ensureDefaultSources  
- Будущий source-first подход  

### 2.4 AI Layer
- System Prompt  
- Режимы short / normal / long  
- Правило минимально достаточного ответа  
- Формирование финального ответа  

### 2.5 Task Engine
- Таблица tasks  
- Cron Render (robot-layer)  
- Базовые типы задач: manual, price_monitor  
- Robot-layer — бесплатные операции без OpenAI  
- Будущее: отчёты, мониторинг, автоматика
